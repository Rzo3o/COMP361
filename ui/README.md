# UI Module (Pygame Edition)

This module manages the game window, input loop, and user interface elements using **Pygame**.

**Files:**
- `game_window.py`: Contains the main game loop (`while running:`), event polling (`pygame.event.get()`), and the update/draw cycle.
- `button.py`: A custom `Button` class for handling clickable UI elements in Pygame (detection of mouse hover/click).
